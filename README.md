# cruise-product-agent · Claude Code 插件

智能助手为您提供邮轮产品的快速查询服务，随时随地获取信息。

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/cruise-product-agent-claude-plugin
/plugin install cruise-product-agent@cruise-product-agent-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **cruise-product-agent**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
